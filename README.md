Hospital Management System

📌 Project Description

A console-based Hospital Management System developed using Java.
This project helps manage hospital operations such as patient records, doctor details, and appointment scheduling.

🚀 Features

* Add new patients
* Add doctor information
* Book appointments
* View patient records
* Simple menu-driven console interface

🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Java Collections

📂 Project Structure

src/

* Main.java
* Patient.java
* Doctor.java
* Appointment.java

🎯 Learning Objectives

This project demonstrates the use of:

* Classes and Objects
* Encapsulation
* Inheritance
* Basic Java program structure
